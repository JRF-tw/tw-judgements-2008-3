臺灣板橋地方法院刑事判決             97年度簡上字第233號
上  訴  人
即  被  告  甲○○
上列上訴人因詐欺案件，不服本院96年度簡字第8213號中華民國
97年1 月31日第一審判決（聲請案號：96年度偵字第24529 號）
，提起上訴，本院管轄之第二審合議庭判決如下：
    主  文
上訴駁回。
    事實及理由
一、本案經本院審理結果，認第一審簡易判決認被告甲○○犯刑
    法第30條、第339 條第1 項之幫助詐欺取財罪，判處有期徒
    刑四月，如易科罰金以銀元三百元折算一日，減為有期徒刑
    二月，如易科罰金以銀元三百元折算一日，認事用法及量刑
    均無不當，應予維持，並引用如附件第一審判決書記載之事
    實、證據及理由。
二、上訴意旨略以：上訴人即被告甲○○前因機車置物箱遭撬開
    ，失竊銀行存摺及金融卡，遭盜用銀行帳戶而涉詐欺案件，
    有臺北縣政府警察局海山分局江翠派出所報案紀錄，並已經
    臺灣彰化地方法院（上訴狀誤載為「雲林地院」）檢察署檢
    察官以95年度偵字第10663 號、第10927 號不起訴處分在案
    ，何以臺灣板橋地方法院檢察署檢察官重啟偵查起訴，爰提
    起上訴云云。
三、但查，原審判決所認被告之犯罪事實，業據被害人乙○○於
    警詢指證綦詳（此被害人於警詢之陳述，被告就其證據能力
    並未爭執，依刑事訴訟法第159 條之5 規定，得為證據），
    並有被告本件系爭銀行帳戶開戶資料、交易明細、被害人報
    案之警局受理各類案件紀錄表、受理詐騙帳戶通報警示簡便
    格式表、報案三聯單等附卷可稽。被告雖矢口否認上揭幫助
    詐欺犯行，辯稱：伊本件臺灣中小企業銀行土城分行帳戶，
    係於95年1 月間，在臺北縣板橋市○○街全家便利商店前，
    因所騎機車置物箱遭人撬開，而失竊皮包，內有伊之身分證
    、健保卡、駕照及本件銀行存摺、金融卡等物，伊並有向該
    轄區之江翠派出所報案，之後伊約隔二週就去辦理補發身分
    證、健保卡、駕照，再隔1 個月伊才去銀行辦理掛失，因本
    件銀行帳戶係伊去中和一家電子工廠工作時，開戶做為薪資
    轉帳使用，約使用1 、2 個月就沒做了云云，然查本件被告
    上開臺灣中小企業銀行土城分行帳戶係於95年2 月8 日新開
    戶，核與被告上開所稱遺失時間已有不符，其次被告上開銀
    行帳戶於同年月16日即遭詐欺集團作為詐騙被害人匯款之人
    頭帳戶使用，與被告上開所述使用1 、2 個月之情亦不合，
    再者被告聲稱本件上開銀行帳戶係其至中和某電子工廠工作
    時開戶作為薪資轉帳使用云云，然其既無法明確說明工作之
    工廠名稱所在，且其既非居於臺北縣土城市，工作之工廠亦
    非在臺北縣土城市，卻在臺北縣土城市○○路開立本件銀行
    帳戶作為薪資轉帳之帳戶，顯亦與常情有違，此外經本院函
    詢各相關單位查覆結果，被告重型機車駕照係於95年3 月2
    日補發，被告健保卡係於95年3 月間申請補發，被告身分證
    則係於95年2 月20日申請補發，被告上開臺灣中小企業銀行
    土城分行帳戶係於95年2 月23日申請掛失，至臺北縣警察局
    海山分局則查無被告於95年1 至2 月間有向該分局江翠派出
    所報案紀錄，以上諸情分別有交通部公路總局臺北區監理所
    97年6 月16日北監駕字第0970030885號函、中央健康保險局
    97 年6月12日健保承字第0970027926號函、臺北縣三峽鎮戶
    政事務所97年6 月10日北縣峽戶字第0970002516號函、臺灣
    中小企業銀行土城分行97年6 月12日97土城字第1420970140
    號函、臺北縣政府警察局海山分局97年6 月23日北縣海偵字
    第0970025594號函等在卷可按，亦核與被告上開所述情節未
    盡相符，綜上諸情，參互印證，足見被告上開所辯，要屬卸
    責之詞，應不足採，是本件事證明確，被告上揭犯行堪以認
    定。原審判決認事用法，核無不合，另衡諸被告本件犯罪動
    機、手段、所生危害及犯後態度等一切情狀，原審判決量刑
    亦屬妥適。另被告上開上訴意旨所述理由，指稱本件犯罪事
    實前已經不起訴處分在案，係重行起訴云云，經核被告所指
    臺灣彰化地方法院檢察署檢察95年度偵字第10663 號、第10
    927 號不起訴處分所敘警方報告事實，係就被告開立之中國
    農民銀行三峽分行帳戶部分所為偵查結果，與本件被告上開
    涉案銀行帳戶不同，所涉犯罪事實不同，是被告上開所指，
    容有誤解，顯亦不足採，從而其執以提起上訴，當顯無理由
    ，自應予駁回。
據上論斷，應依刑事訴訟法第455 條之1 第1 項、第3 項、第36
8 條、第373 條，判決如主文。

│    之貨幣單位原為銀元，舊刑法第33條第5 款規定：「罰金：  │
│    1 元（銀元）以上。」，而依罰金罰鍰提高標準條例規定，  │
│    就72年6 月26日前修正之刑法部分條文罰金數額提高2 至10  │
│    倍，其後修正者則不提高倍數，並依現行法規所定貨幣單位  │
│    折算新臺幣條例規定，以銀元1 元折算新臺幣3 元。而新刑  │
│    法第33條第5 款規定：「罰金：新臺幣1000元以上，以百元  │
│    計算之。」，新刑法第33條第5 款所定罰金貨幣單位經修正  │
│    為新臺幣後，刑法分則各罪所定罰金刑之貨幣單位亦應配合  │
│    修正為新臺幣，為使刑法分則編各罪所定罰金之最高數額與  │
│    刑法修正前趨於一致，乃增訂刑法施行法第1條 之1 ：「中  │
│    華民國94年1 月7 日刑法修正施行後，刑法分則編所定罰金  │
│    之貨幣單位為新臺幣。94年1 月7 日刑法修正時，刑法分則  │
│    編未修正之條文定有罰金者，自94年1 月7 日刑法修正施行  │
│    後，就其所定數額提高為30倍。但72年6 月26 日 至94年1   │
│    月7 日新增或修正之條文，就其所定數額提高為3 倍。」。  │
│    從而，刑法分則編各罪所定罰金刑之最高數額，於上開規定  │
│    修正後並無不同，惟新刑法第33條第5 款所定罰金刑最低數  │
│    額，較之修正前提高，自以舊刑法第33條第5 款規定有利於  │
│    被告。(二)至於被告行為後刑法第30條之部分雖已修正施行，  │
│    然舊刑法第30條係規定：「幫助他人實行犯罪行為者，為幫  │
│    助犯。雖他人不知幫助之情者，亦同。幫助犯之處罰，得按  │
│    正犯之刑減輕之。」，而新刑法第30條則規定：「幫助他人  │
│    犯罪者，為從犯。雖他人不知幫助之情者，亦同。從犯之處  │
│    罰，得按正犯之刑減輕之。」僅係將幫助犯規定為法理之明  │
│    文化，不涉罪刑實質內容之變更，非屬法律之變更，自不生  │
│    新舊法比較之問題，應逕適用裁判時之刑法第30條之規定。  │
│四、核使用被告上開臺灣中小企銀帳戶之詐欺集團成員意圖為自  │
│    己不法之所有，以詐術使人將本人之物交付，所為係犯刑法  │
│    第339 條第1 項之罪。而被告以提供上開帳戶存摺等物予詐  │
│    欺集團使用之行為，則係犯刑法第30條第1 項、第339 條第  │
│    1 項幫助詐欺取財罪。又被告以幫助之意思，參與構成要件  │
│    以外之行為，為幫助犯，爰依刑法第30條第2 項規定，依正  │
│    犯之刑減輕之。爰審酌被告將上開帳戶存摺等物件交付他人  │
│    ，助長他人犯罪，增加被害人尋求救濟及治安機關查緝犯罪  │
│    之困難，危害被害人財產安全及社會治安，兼衡被告素行、  │
│    犯罪手段、對被害人財產所生危害程度，及其犯後否認犯行  │
│    之態度等一切情狀，量處如主文所示之刑。而刑法第41條第  │
│    1 項前段規定：「犯最重本刑為五年以下有期徒刑以下之刑  │
│    之罪，而受六個月以下有期徒刑或拘役之宣告，因身體、教  │
│    育、職業、家庭之關係或其他正當事由，執行顯有困難者，  │
│    得以一元以上三元以下折算一日，易科罰金。」，而被告行  │
│    為時之易科罰金折算標準，依95年7 月1 日修正條文施行前  │
│    罰金罰鍰提高標準條例第2 條前段之規定（該條業經總統於  │
│    95 年5月17日公布刪除，並自95年7 月1 日失效），就其原  │
│    定數額提高為100 倍折算1 日，則被告行為時之易科罰金折  │
│    算標準，應以銀元100 元至300 元折算1 日，經折算為新臺  │
│    幣後，係以新臺幣300 元至900 元折算為1 日；惟新刑法第  │
│    41 條 第1 項前段則規定：「犯最重本刑為五年以下有期徒  │
│    刑以下之刑之罪，而受六個月以下有期徒刑或拘役之宣告者  │
│    ，得以新臺幣一千元、二千元或三千元折算一日，易科罰金  │
│    。」，則比較修正前後之易科罰金折算標準，應以舊刑法第  │
│    41條第1 項前段規定較有利於被告，爰依此條文規定諭知如  │
│    易科罰金之折算標準，以資懲儆。又中華民國九十六年罪犯  │
│    減刑條例業經立法院制定，並業經總統於96年7 月4 日公布  │
│    ，自96年7 月16日起生效，本件被告犯罪時間係在96年4 月  │
│    24日以前，合於上開減刑條例第2 條第1 項第3 款之規定，  │
│    且非同條例第3 條所定不得減刑之罪，是併依上開減刑條例  │
│    規定減其二分之一之刑，並諭知易科罰金之折算標準。      │
